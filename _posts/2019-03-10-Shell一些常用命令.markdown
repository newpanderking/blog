### shell 常用命令

* `ctrl + r` : 查找历史执行命令，输入关键字匹配。
* `ctrl + p` : 前一条命令。
* `ctrl + n` : 后一条命令。
* `ctrl + c` : 如果命令尚未执行完毕，终止正在执行的命令;如果命令尚未执行，取消命令。 
* `ctrl + a` : 移到光标命令行开始出。 
* `ctrl + e` : 移到光标命令行结束出。 
* `ctrl + h` : 向前移动一个字符。 
* `ctrl + d` : 向后移动一个字符。 
* `ctrl + b` : 光标向前移动。 
* `ctrl + f` : 光标向后移动。 
* `ctrl + k` : 删除光标到命令行行尾。 
* `ctrl + u` : 删除光标到命令行开始。 
* `ctrl + b` : 光标向前移动。 
* `ctrl + b` : 光标向前移动。
